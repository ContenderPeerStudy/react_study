HTML (Hyper Text Markup Language)  
- 마크업 언어의 한 종류  
- 마크업이란 문서나 데이터를 처리하기 위해 문서에 추가되는 정보를 의미합니다.  
  마크업 정보를 표현하기 위한 언어를 마크업 언어라 합니다. 
- 주로 웹에서만 사용하였으나 요즘에는 웹 브라우저를 사용해서 데스크탑용  
  앱을 만들수 있는 기술들도 많이 발전   
 
-> 웹사이트의 뼈대를 구성하기 위해 사용하는 마크업 언어

- Tag(태그)라고 부르는 것을 이용하여 웹사이트의 구조를 만들고 웹사이트에
  들어갈 내용들을 채워 넣는다. <html> </html>
- 태그는 열었으면 꼭 닫아줘야 함!
